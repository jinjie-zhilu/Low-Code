# Low-Code
字节跳动青训营结营大项目Low-Code

技术栈：

- Vue 3
- TypeScript
- Vite
- Element Plus

## 功能点进展

- 顶部工具栏样式
- 左侧组件列表样式及拖拽功能
- 中间画布样式及渲染功能
- 右侧属性设置样式
- 清空画布功能
- 设置画布属性
- 主题切换/昼夜模式
